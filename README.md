Ce projet consiste en le développement d’une galerie dynamique permettant d’afficher, filtrer et administrer les travaux de l’architecte Sophie Bluel, avec une interface dédiée à l’administrateur.
L’objectif principal était de créer une galerie entièrement dynamique, synchronisée avec un back-end, et offrant une interface d’administration sécurisée pour permettre la gestion des médias (ajout et suppression).

Objectifs du projet:

- Récupérer et afficher les projets depuis une API back-end.

- Implémenter un système de filtres par catégories.

- Créer une interface d’administration avec authentification.

Permettre à l'administrateur :

- d’ajouter de nouveaux médias en temps réel via un formulaire et FormData

- de supprimer des projets directement depuis la modale d’édition

Mettre à jour le DOM dynamiquement sans rechargement de page.

Stack technique:

-HTML

-CSS

-JavaScript

-Fetch API

-Node.js (back-end fourni)

Compétences développées

-Gestion des événements utilisateurs

-Requêtes API et gestion des réponses

-Envoi de formulaire via FormData

-Authentification via token et stockage sécurisé

-Mise à jour dynamique du DOM (ajouts/suppressions)

-Création et gestion de modales

-Filtrage côté front en fonction d’une donnée API

Gestion d’un token d’authentification

Manipulation avancée du DOM
